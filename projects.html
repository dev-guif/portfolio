[N8N/Automação] - Automação de Ofertas – Shopee

Desenvolvi uma automação para monitorar produtos na Shopee e enviar as melhores ofertas diretamente para um grupo de WhatsApp. O sistema identifica produtos com maiores descontos e envia as promoções de forma rápida e organizada, garantindo que os usuários recebam apenas oportunidades relevantes. Além disso, o projeto conta com uma plataforma web para acompanhamento de todos os produtos lançados no grupo, permitindo monitorar histórico, descontos e métricas de engajamento.

  • Busca automatizada dos produtos com maiores descontos na Shopee.
  • Filtragem por categorias e relevância das ofertas.
  • Envio automático das ofertas para um grupo de WhatsApp.
  • Atualizações em tempo real, garantindo que as promoções cheguem rapidamente aos usuários.
  • Plataforma web para acompanhamento completo de todos os produtos e histórico de ofertas.
  • Redução de tempo e esforço manual para acompanhar promoções online.

Tecnologias utilizadas:

  • n8n para orquestração do fluxo de coleta e envio.
  • APIs da Shopee para busca de produtos e preços.
  • Evolution API para envio das mensagens no WhatsApp.
  • Filtragem e ranking automatizado dos produtos mais vantajosos.
  • Plataforma web para visualização, histórico e métricas de engajamento.

Resultado: os usuários recebem as melhores ofertas da Shopee diretamente no WhatsApp, com precisão, rapidez e relevância, enquanto a plataforma web permite acompanhar e analisar todos os produtos lançados.

[N8N/Agente de IA] - Agente de Agendamento Inteligente

Imagine ter um assistente que agenda, edita, cancela, envia lembretes, avisa o time e ainda tem base de conhecimento — tudo isso de forma automática via WhatsApp. Desenvolvi este agente para simplificar e automatizar o fluxo de compromissos de empresas e profissionais que trabalham com atendimento.

  • Recebe pedidos de agendamento por áudio ou texto via WhatsApp.
  • Cria, edita e cancela eventos no Google Calendar automaticamente.
  • Realiza follow-up com os participantes e envia avisos personalizados.
  • Notifica em grupos quando um novo lead é agendado.
  • Consulta e responde com base em informações internas da empresa.
  • Suporte a múltiplos atendentes ou agendas.

Tecnologias utilizadas:

  • n8n como motor de automações.
  • Google Calendar API para manipulação de eventos.
  • OpenAI (ChatGPT) e DeepSeek como LLMs para interpretação de mensagens.
  • Supabase para armazenamento de leads e compromissos.
  • Google Drive + Vector Store para base de conhecimento.
  • Evolution API (WhatsApp) e webhooks personalizados para integração.

Resultado: fluxo multiagente robusto, 24/7, gerando economia de tempo, aumento na organização e melhor experiência para leads e clientes.

  • Home: A página principal oferece um acompanhamento quase em tempo real da demanda e fila de chamados para o setor. Este projeto demonstra a capacidade de criar soluções de monitoramento eficientes e intuitivas, facilitando a gestão e análise das atividades de suporte setorial em informática.
